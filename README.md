# Fastify API Base

Projeto base para estudos de **Node.js + TypeScript + Fastify**, focado em boas práticas,
organização de código e ferramentas modernas do ecossistema backend.

## Stack

- Node.js
- TypeScript
- Fastify
- Zod (validação)
- ESLint + Prettier
- pnpm

env.ts 
server.ts
